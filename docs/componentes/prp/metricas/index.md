# Métricas

No PRP, as métricas transformam o fluxo de trabalho em dados analisáveis.

Essa camada existe para permitir leitura clara da operação, identificar problemas recorrentes e apoiar decisões com base em evidência, não percepção.

---

## Princípio

O que não é medido não pode ser melhorado.

- toda operação gera dados
- dados devem ser estruturados
- métricas devem refletir o comportamento real do sistema

O objetivo não é controle individual, mas entendimento coletivo da operação.

---

## Estrutura

As métricas do PRP são organizadas em quatro dimensões:

- fluxo
- prioridade
- bloqueio
- desperdício

Essa separação permite analisar diferentes aspectos do trabalho.

---

## Fluxo

Mede como as demandas percorrem o sistema.

- tempo médio de execução
- tempo em cada etapa
- quantidade de tarefas concluídas

Esses dados ajudam a identificar gargalos e melhorar previsibilidade.

---

## Prioridade

Mede o comportamento das classificações de nível.

- volume por nível (P0, P1, P2, P3)
- frequência de demandas críticas
- mudanças de prioridade ao longo do tempo

Esses indicadores mostram se o sistema está sendo usado corretamente.

---

## Bloqueio

Mede interrupções no fluxo.

- quantidade de tarefas bloqueadas
- tempo médio em bloqueio
- principais causas de bloqueio

Esses dados evidenciam problemas de comunicação e dependência.

---

## Desperdício

Mede esforço que não gerou resultado.

- demandas canceladas
- retrabalho
- mudanças de escopo

Essa dimensão é essencial para identificar falhas na priorização.

---

## Registro

As métricas devem ser coletadas de forma simples e consistente.

- uso correto de status no board
- registro de bloqueios
- atualização de prioridade
- finalização adequada das tarefas

A qualidade dos dados depende da consistência do uso do sistema.

---

## Análise

Os dados coletados devem ser analisados periodicamente.

- identificação de padrões
- comparação entre períodos
- leitura de tendências

Essa análise permite evolução contínua do processo.

---

## Uso

As métricas devem ser utilizadas para:

- melhorar o fluxo de trabalho
- ajustar priorização
- reduzir desperdício
- apoiar decisões estratégicas

Não devem ser usadas para controle individual ou avaliação isolada de desempenho.

---

## Papel

Dentro do PRP, as métricas são responsáveis por:

- transformar operação em informação
- dar visibilidade para gestão
- identificar problemas estruturais
- orientar melhorias contínuas

Sem métricas, o sistema depende de percepção e opinião.

---

## Objetivo

O uso de métricas busca garantir que:

- decisões sejam baseadas em dados
- o time evolua continuamente
- gargalos sejam identificados com clareza
- a operação se torne cada vez mais eficiente

Aplicar corretamente as métricas é o que permite ao PRP sair do nível operacional e atuar como ferramenta estratégica.
