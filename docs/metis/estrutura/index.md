# Estrutura do METIS

O METIS é organizado em camadas que definem sua lógica de funcionamento e sua capacidade de adaptação.

## Framework

O METIS é a camada principal. Ele define a arquitetura geral, os princípios e a relação entre seus componentes.

## Metodologias

As metodologias são responsáveis por estruturar o trabalho contínuo.

### Adaptive Flow System

O AFS é a metodologia do METIS voltada para fluxo contínuo, organização operacional e adaptação progressiva.

## Protocolos

Os protocolos são acionados em cenários específicos que exigem resposta orientada e rápida.

### Priority Response Protocol

O PRP é o protocolo do METIS voltado para gestão de crise, priorização e tomada de decisão sob pressão.

## Relação entre as camadas

O METIS organiza o sistema.
O AFS sustenta o fluxo.
O PRP atua quando o fluxo precisa ser protegido, redirecionado ou interrompido com clareza.
